The variables of the dataset in the file "summ_by_activity_subject.txt" are:

Activity
- Activity performed, one of: WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING.

Subject
- Code of the volunteer who performed the activity: from 1 to 30.

All the other variables are actually average (mean) of the base variable coming from the: 
"Human Activity Recognition Using Smartphones Dataset - Version 1.0" dataset.
I.e. each column for a given row (pair Activity-Subject) is the average of the corresponding variable based on the total of Activities (of the same kind) performed by a single Subject.
